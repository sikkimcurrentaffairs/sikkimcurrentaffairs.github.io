# Sikkim PSC Preparation Hub

Welcome to the **Sikkim PSC Preparation Hub** repository! 

This is a comprehensive, free, and open-source platform tailored for aspirants preparing for the Sikkim Public Service Commission (SPSC) exams. Our goal is to provide quality study materials, current affairs, mock tests, and AI-powered notes without any subscription fees.

## 🌟 Features
- **Sikkim Current Affairs & Static GK**: Monthly updated quizzes and highlights specific to Sikkim.
- **National & International Affairs**: Stay updated with broader monthly current affairs.
- **General Studies**: Deep dive into subjects like History, Geography, Polity, Biology, Physics, and more.
- **English & Vocabulary**: Dedicated sections to enhance your language skills.
- **Notes (AI)**: Smart AI-generated notes and flashcards for quick revision.
- **Mock Tests & Question Banks**: Practice with extensive question sets and past year papers (PYQs).
- **Interactive UI**: A beautiful, neomorphic design with seamless light/dark mode support.
- **User Dashboard**: Track your quiz progress, review weak areas, and maintain study streaks with Firebase integration.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Tailwind CSS, Custom Neomorphic CSS
- **Backend & Auth**: Firebase Authentication (Google Sign-In), Cloud Firestore
- **Hosting**: GitHub Pages
